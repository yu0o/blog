## Editor

在 ps 网站直接 **ctrl s** 会直接保存一份本地文件

然后在 editor 直接打开这个文件，选择低于 5.2 版本，进入后切换 Cubism 4.0 SDK。

![](/blogs/live2d-note/e25e0390577882f7.webp)

### 制作理论

我们这里只实现角色上下左右看，所以只需要关注参数的 **角度XY**, 这里注意 x: -30 是向左看，y:-30 是向下看，点击参数左边的连接 icon 可以 **2维** 操控。

![](/blogs/live2d-note/7da530d9c3f832d2.webp)

**教程**中，都是使用给每个图片设置变形器的，但是这里的要求简单，就不创建了，一个个创建变形器也比较麻烦。

这个软件的逻辑是，不管你用不用变形器等，只要打上关键帧，你的操作就会被记录，你的更改也只能在关键帧的点上改。我这里就直接为所有图片创建新的网格（下半身不动就不用）。

![](/blogs/live2d-note/d29426c1f68bba64.webp)

网格选择“标准”，然后直接覆盖，默认的网格是方形，不方便变形。

### 关键帧

制作的时候只用单独制作角度 x，y，最后选择两个参数进行“**四角形状合成**”。没有合成之前四个角落是记录的初始值。

也是选择所有需要动的图片，创建关键帧（参数下面，3个点的绿图标）

**右键**直接选中帧，x -30，先将眼睛、眉毛、鼻子、嘴巴往左移动，耳朵合适缩放，脸部的画缩放加上一些阴影变形，需要直接调整里面的点，让阴影跟着运动拉伸。（如：脸部右下角的点变化）

![](/blogs/live2d-note/db79f5eb2a741529.webp)

![](/blogs/live2d-note/c73ed7e388028a09.webp)

头发也是，让它有个较为真实的角度变换。

细调需要耐心慢慢一遍一遍尝试。我这里就不再赘述。

### 镜像动作

x-30 之后，脸部的动画可以直接镜像到右边去，选择觉得可以直接镜像的图片，然后处于 -30 的帧下，点参数的右侧菜单选择“**动作镜像**”

![](/blogs/live2d-note/759d5a1a17eec4bc.webp)

### 如何还原变换

很可能操作不当，需要还原，他这里没有直接的还原命令，选择不要的变形，拖动帧回到 0 处，进行删除帧的操作，然后再创建帧。

## 导出

导出需要选择“运行时文件”，但是直接导出会提示没有“纹理集”，所以需要在 菜单-建模-纹理-纹理集 生成一份文件。

再进行*运行时导出*就行，导出的 png 图片可以压缩一下。
